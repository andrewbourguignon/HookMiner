# HookMiner Skill

## Description
Expert system for mining proven viral hooks from Instagram and TikTok profiles. Scans video handles, filters by view counts, and transcribes hooks using LLMs.

## Features
- Profile Scanning (Instagram/TikTok)
- View Count Filtering
- LLM hook extraction
- CSV Export

## Usage
- "Run HookMiner on @handle"
- "Update HookMiner database for @handle"
- "Show me the top hooks from HookMiner for @handle"

## Implementation Details
- Main entry: `app.py` or `main.py`
- Location: `.agent/skills/hook-miner/`
- Dependencies: `pip install -r requirements.txt`
